Get the archive

Download the 800,000 Drum Percussion MIDI Archive from the Internet Archive and unzip it somewhere on your machine.

2

Point to it

Enter the path to the top-level archive folder (the one containing genre subfolders like Jazz, Rock, Blues, etc.).

3

Build the database

This scans every MIDI file and extracts musical features (tempo, time signature, instruments, beat positions). The curated folders (~19k files) take about 15 seconds. You only need to do this once — the database is saved to disk and will be used on future runs unless you delete drums.db.
